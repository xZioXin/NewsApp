Про додаток

Новинний додаток - це веб-застосунок для публікації та обговорення новин. Додаток дозволяє користувачам:

  • Переглядати останні новини
  • Створювати власні новини
  • Коментувати новини інших користувачів
  • Редагувати свої новини

Для адміністраторів доступні додаткові функції:

  • Модерація новин
  • Видалення неправильного контенту

Інструкція користування:

  1. Зареєструйтеся або увійдіть у систему
  2. На головній сторінці переглядайте новини
  3. Для створення новини натисніть "Створити новину"
  4. Заповніть форму та збережіть новину
  5. У розділі "Мої новини" ви можете керувати своїми публікаціями
